# NeoScan AI — Website

Full-stack web application for NeoScan AI neonatal screening system.

## Structure
- Frontend: index.html — NeoScan AI landing page
- Backend: FastAPI Python server — AI model inference API

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python, FastAPI
- AI Models: PyTorch (.pth), Keras (.keras)
